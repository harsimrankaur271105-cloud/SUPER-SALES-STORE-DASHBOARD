Interactive Power BI dashboard for Super Store sales analysis with regional insights, customer segmentation, and 15-day sales forecasting.
Forecast (Power BI)
This project presents an interactive Power BI dashboard built on Super Store sales data to analyze business performance and generate short-term sales forecasts.
Project Overview
The dashboard provides a comprehensive view of sales, profit, quantity, and customer segments, helping businesses make data-driven decisions. It also includes a 15-day sales forecast using historical trends.
 Key Features
 1. Sales Performance Overview
Total Sales: 522K+
Total Quantity Sold: 7,298
Total Profit: 68K
Average Sales Value: 4
Provides a quick snapshot of overall business performance.
 2. Time-Based Analysis
Monthly Sales Trend (YoY) – Tracks growth patterns across months.
Monthly Profit Trend (YoY) – Identifies profit fluctuations and seasonality.
3. Category & Sub-Category Insights
Sales categorized into:
Office Supplies
Technology
Furniture

 4. Regional & Segment Analysis
Sales distribution by Region (Central, East, South, West)
Customer segmentation:

 5. Shipping Mode Analysis
Sales comparison across:

Second Class
First Class
Same Day
 6. Profit Distribution
Visual representation of profit contribution across different categories and segments.
 7. State-wise Sales Analysis
Highlights top-performing states such as:

 Sales Forecast (15 Days)
Uses historical sales data to predict future trends.
Interactive timeline visualization for better trend analysis.
Helps in planning inventory and business strategy.
